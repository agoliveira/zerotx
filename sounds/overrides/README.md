# Audio overrides

Drop hand-recorded WAV or MP3 files here, named the same as a dictionary
track. The build script copies them over the generated audio at the end of
each language pass, so a hand-recorded clip always wins.

```
overrides/en/armed.wav
overrides/en/failsafe.wav
overrides/pt/armed.wav
```

Files in this directory are committed to git and shipped with the project.
By contributing overrides, you licence your recording under the project's
GPLv3 terms.
